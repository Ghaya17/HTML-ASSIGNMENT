HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items. HTML elements are delineated by tags, written using angle brackets. Tags such as directly introduce content into the page.


HTML elements tell the browser how to display the content. For example, an element might label a piece of content as a heading, paragraph, or link.
